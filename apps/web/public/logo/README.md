# Logo Files

## Upload Your Logo Here

Place your logo file in this directory.

## Recommended Files:

- **logo.png** - Main logo (recommended)
- **logo.svg** - Vector logo (scalable, best quality)
- **logo-white.png** - White version for dark backgrounds
- **logo-dark.png** - Dark version for light backgrounds

## Recommended Sizes:

- **Full logo**: 200x60px (or proportional)
- **Icon only**: 40x40px or 60x60px
- **Format**: PNG with transparency (or SVG)

## File Naming:

Use one of these names:
- `logo.png` (main logo)
- `vannilli-logo.png`
- `brand-logo.png`

Then update the navigation component to use your filename.
